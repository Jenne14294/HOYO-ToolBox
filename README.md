# 🧰 HOYO ToolBox

一個簡單的 HoYoverse 小工具，專為《原神》、《崩壞：星穹鐵道》與《絕區零》玩家打造，讓你輕鬆管理多個帳號並精準掌握遊戲數據！

---

## ✨ 可用功能 (Features)

### 🎯 抽卡紀錄
告別手動計算，輕鬆掌握你的保底與出金數據：
- **多遊戲支援**：完整支援《原神》、《崩壞：星穹鐵道》、《絕區零》。
- **導出歷史紀錄**：支援將遊戲內的抽卡紀錄匯出儲存，方便永久保存與跨工具讀取。
- **計算運氣**：自動幫你計算已墊抽數、平均出金率、不歪率與歷史歐非狀態。

### 🛠️ HoYoLAB 工具箱
無縫接軌官方社群數據，日常功能一鍵搞定：
- **即時便籤**：快速查看樹酯/開拓力/電量是否滿溢，以及派遣任務進度。
- **收入月曆**：追蹤每個月的星瓊/原石/菲林獲取量與來源。
- **戰績**：詳細檢視帳號擁有的角色練度與深淵/忘卻之庭/式輿防衛戰通關紀錄。
- **每日簽到**：輕鬆完成官方每日簽到，不再忘記領獎勵。
- **兌換碼**：快速兌換前瞻直播或活動發放的序號。
- **互動地圖**：內建官方互動地圖捷徑（*註：目前僅限原神和崩鐵*）。

### ⚙️ 額外功能
專為多開玩家設計的便利功能：
- **儲存帳號**：安全記錄並綁定你的遊戲 UID 與設定。
- **多帳號切換**：支援一鍵切換不同帳號的資料面板，無須反覆登入登出。

## 🚀 快速開始 (Getting Started)

> **💡 提示**：請確保你的電腦已安裝 Python 3.8 或以上版本。

1. **複製專案到本地**
   ```bash
   git clone [https://github.com/Jenne14294/HOYO-ToolBox.git](https://github.com/Jenne14294/HOYO-ToolBox.git)
   ```
2. **安裝必要的依賴套件**
   ```bash
   pip install -r requirements.txt
   ```

3. **執行主程式**
   ```bash
   python gui.py
   ```
